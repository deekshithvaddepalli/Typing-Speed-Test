# CVIP-PYTHONDEVELOPMENT-PHASE2-NORMALTASK

A simple desktop-based Typing Speed Test application developed using Python and Tkinter.
This project helps users measure their typing speed by calculating Words Per Minute (WPM) based on how quickly and accurately a displayed word is typed.
